Описание / Beschreibung

Это простая консольная RPG-игра, написанная на Python. Игрок создаёт профиль, выбирает героя, исследует уровни, сражается с врагами и получает награды. Игра реализована через словари и функции, поддерживает прогресс игрока.
Dies ist ein einfaches Konsolen-RPG-Spiel in Python. Der Spieler erstellt ein Profil, wählt einen Helden, erkundet Levels, kämpft gegen Gegner und erhält Belohnungen. Das Spiel verwendet Dictionaries und Funktionen und unterstützt den Fortschritt des Spielers.

Особенности / Funktionen

Создание и редактирование профиля игрока / Erstellen und Bearbeiten eines Spielerprofils

Выбор героя с разными характеристиками (воин, ассасин, танк) / Heldenauswahl mit unterschiedlichen Eigenschaften (Krieger, Assassine, Tank)

Пошаговые сражения с врагами / Rundenbasierte Kämpfe gegen Gegner

Система уровней с постепенным открытием новых / Level-System mit schrittweiser Freischaltung neuer Levels

Награды за победу: золото и опыт / Belohnungen für Siege: Gold und Erfahrung

Как играть / Spielanleitung

Запустите скрипт python main.py / Skript starten: python main.py

Создайте профиль и введите имя и возраст / Profil erstellen und Name sowie Alter eingeben

Выберите героя / Helden auswählen

Исследуйте уровни и сражайтесь с врагами / Levels erkunden und gegen Gegner kämpfen

Смотрите статистику игрока в меню профиля / Spielerstatistik im Profil-Menü ansehen

Требования / Anforderungen

Python 3.x
